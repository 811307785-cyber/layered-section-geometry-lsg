# Aerospace-Space 航空航天大尺度测绘模块
## 模块定位
针对大尺度航空航天遥感场景优化的LSG重建模块，解决大气折射、地球曲率导致的远距离畸变问题。

## 核心功能
1. GeoTIFF等标准遥感影像数据解析
2. 卫星影像条带噪声、辐射不均自适应消除
3. 大气折射、地球曲率大尺度畸变校正
4. 支持百公里级跨度地形连续顺滑重建

## 依赖
- 底层内核：Common-LSG v0.1+
- 第三方依赖：numpy, gdal
